============================================================
    檔案自動分類工具 V2.0 - 使用說明 / User Manual
============================================================

(繁體中文)
歡迎使用 V2.0！本工具可以根據您建立的多組「設定檔」，
自動從來源資料夾中，將符合條件的檔案複製到指定的目標資料夾，
讓您輕鬆應對多種工作情境，一鍵完成複雜的檔案整理任務。

(English)
Welcome to V2.0! This tool can automatically copy files from source folders 
to specific destinations based on multiple "Profiles" that you create.
It allows you to effortlessly manage various workflows and complete 
complex file organization tasks with a single click.


【快速開始 | Quick Start】
------------------------------------------------------------

1. 設定檔管理 (Profile Management) <-- V2.0 核心功能!
   (繁體中文)
   - 「設定檔」就像遊戲存檔，您可以為不同的工作情境（例如：「BIM 每週上傳」、「個人照片整理」）建立獨立的設定。
   - **儲存設定**：完成 A, B, C 區塊的設定後，點擊「另存新檔...」來建立您的第一個設定檔。之後若有修改，點擊「儲存」即可。
   - **載入設定**：從頂部的下拉選單中選擇您想用的設定檔，然後點擊「載入」，所有設定會立刻切換。
   (English)
   - A "Profile" is like a game save. You can create separate settings for different workflows 
     (e.g., "Weekly BIM Upload," "Personal Photo Sorting").
   - **Saving a Profile**: After configuring Sections A, B, and C, click "Save As..." to create your first profile. 
     To update it later, just click "Save".
   - **Loading a Profile**: Select a profile from the top dropdown menu and click "Load" to instantly switch all settings.


2. 設定「來源」資料夾 (Set Source Folders)
   (繁體中文)
   - **新增來源**：點擊「新增來源」逐一選擇資料夾。
   - **批次新增**：點擊「批次新增」，您可以一次性貼上多個資料夾路徑（一行一個）。
   - **介面拉伸**：A 區和 B 區中間的分隔線可以上下拖曳，方便您檢視大量路徑。
   (English)
   - **Add Source**: Click "Add Source" to add folders one by one.
   - **Batch Add**: Click "Batch Add" to paste multiple folder paths at once (one path per line).
   - **Resizable View**: You can drag the separator line between Section A and B to resize the view for many paths.


3. 設定「分類規則」 (Set Classification Rules)
   (繁體中文)
   - 在 B 區塊，點擊「新增規則」來建立分類條件。
   - **規則名稱**：為規則取一個好記的名字。
   - **檔名包含**：輸入檔案名稱的關鍵字。
   - **日期篩選**：選擇 `修改日期晚於 (Modified After)` 等條件。
   - **目標資料夾**：選擇符合規則的檔案要被複製到哪裡。
   (English)
   - In Section B, click "Add Rule" to create your classification criteria.
   - **Rule Name**: Give your rule a descriptive name.
   - **Filename Contains**: Enter a keyword the filename must contain.
   - **Date Filter**: Choose a condition like `Modified After`.
   - **Destination Folder**: Select where the matching files should be copied.


4. 設定「衝突」資料夾 (Set Conflict Folder)
   (繁體中文)
   在 C 區塊，設定一個專門的資料夾。如果一個檔案同時符合多條規則，它就會被複製到這裡。
   (English)
   In Section C, set a dedicated folder. If a file matches multiple rules, it will be copied here.


5. 開始執行！ (Execute!)
   (繁體中文)
   點擊「開始分類檔案」，下方的「狀態日誌」會即時顯示處理進度。
   (English)
   Click "Start Classification". The "Status Log" will show the real-time progress.


【實用技巧 | Pro Tips】
------------------------------------------------------------

*   **語言切換 (Language Switch)**
    (繁體中文)
    在右下方，點擊按鈕可隨時切換「繁體中文 / English」介面。
    (English)
    Click the button in the bottom-right corner to toggle the interface language at any time.

*   **來源管理 (Source Management)**
    (繁體中文)
    除了「移除所選」，現在您也可以點擊「全部移除」來一鍵清空來源列表。
    (English)
    Besides "Remove Selected", you can now click "Remove All" to clear the source list instantly.

*   **快速複製規則 (Copy Rule)**
    (繁體中文)
    要設定類似的規則時，選中一條並點擊「複製所選」，再「編輯」修改即可。
    (English)
    To create similar rules, select one, click "Copy Selected", and then "Edit" the new one.

*   **啟用/停用規則 (Toggle Rule)**
    (繁體中文)
    在規則列表的第一欄，點擊 ✅ 或 ❎ 圖示，可以暫時啟用或停用某條規則。
    (English)
    Click the ✅ / ❎ icon in the first column to temporarily enable or disable a rule.
